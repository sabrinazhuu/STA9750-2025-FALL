Netflix has become a global stage for stories that entertain and connect audiences across cultures. # 8) The movie Red Notice has a runtime of 1 hour and 58 minutes.
# Approximately how many views did it receive in 2021?
# Note that Netflix does not provide the weekly_views values that far back in the past,
# but you can compute it yourself using the total view time and the runtime.
title_8 <- "Red Notice"
rn_runtime <- 1 + 58 / 60

rn_data <- GLOBAL_TOP_10 |>
  filter(
    show_title == title_8,
    year(week) == 2021
  ) |>
  summarise(total_hours_viewed = sum(weekly_hours_viewed, na.rm = TRUE)) |>
  mutate(
    estimated_views = total_hours_viewed / rn_runtime,
    views_millions = round(estimated_views / 1e6, 1)
  )

rn_views <- rn_data |> pull(views_millions)

# 9) How many Films reached Number 1 in the US but did not originally debut there?
# That is, find films that first appeared on the Top 10 chart at, e.g., Number 4 but then became more popular and eventually hit Number 1?
# What is the most recent film to pull this off?

# Find all films that debuted below #1 but hit #1
us_climbers <- COUNTRY_TOP_10 |>
  filter(country_name == "United States", category == "Films") |>
  group_by(show_title) |>
  mutate(
    debuted_rank = weekly_rank[which.min(week)],
    hit_number1 = any(weekly_rank == 1)
  ) |>
  summarise(
    debuted_rank = first(debuted_rank),
    hit_number1 = first(hit_number1),
    most_recent_week = max(week),
    .groups = "drop"
  ) |>
  filter(hit_number1, debuted_rank > 1)

# Count how many films did this
num_climbers <- nrow(us_climbers)

# Get the most recent film
most_recent_climber <- us_climbers |>
  slice_max(most_recent_week, n = 1, with_ties = FALSE)

recent_climber_title <- most_recent_climber |> pull(show_title)
recent_climber_debut <- most_recent_climber |> pull(debuted_rank)

Key Highlights

  • Today, Netflix operates in 94 countries, bringing world-class stories to millions of viewers.

  • The non-English-language film with the longest Global Top 10 run is “All Quiet on the Western Front”, which stayed for 23 weeks.

  • The longest film to appear in the Global Top 10 is “Pushpa 2: The Rule (Reloaded Version)”, running 224 minutes.

  • Across all categories, the leaders in total global hours viewed are:

    • Films (English) - “KPop Demon Hunters” with 523.7 million hours
    • Films (Non-English) - “Society of the Snow” with 235.9 million hours
    • TV (English) - “Stranger Things” with 2968 million hours
    • TV (Non-English) - “Squid Game” with 5048.3 million hours

  • The TV show with the longest Top 10 run in a single country is “Money Heist” in Pakistan, lasting 127 weeks.

  • Nearly every country has over 200 weeks of Netflix service history, except Russia, where operations ended in February 2022.

  • Globally, Squid Game has accumulated 5048.3 million hours viewed across all seasons.

  • With a runtime of 1 hour and 58 minutes, Red Notice received around 201.7 million views in 2021.

  • Not every hit begins at the very top. 45 films started below Number 1 in the U.S. charts and later achieved Number 1 status. The most recent example is “KPop Demon Hunters”, which began at number 6.

  • The TV show with the widest debut reach is “Emily in Paris, 3 Body Problem, Sex Education, Sex/Life, Squid Game: The Challenge, Stranger Things, The Night Agent, Wednesday, Avatar The Last Airbender, DAHMER”, which hit the Top 10 in 94, 93, 93, 93, 93, 93, 93, 93, 92, 92 countries in its first week.
